---
allowed-tools: Read, Write
description: End-of-session wrap-up — writes a summary entry to the session journal.
---

Close out the current working session.

1. Summarize what was done (max 5 bullets).
2. List what is still pending.
3. List any decisions made (link or create ADR if significant).
4. Note the next concrete action.

Append this to `docs/journal/session-notes.md` under today's date.

Format:
```
## YYYY-MM-DD — [short title]

**Done**
- ...

**Pending**
- ...

**Decisions**
- ...

**Next**
- ...
```
